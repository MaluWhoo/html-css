Paragráfos e Quebra de Linha

Um paragráfo pode ser escrito de qualquer maneira, a partir do momento em que essa escrita esteje dentro das tags p e /p

= <p> </p>

Para o caso de precisar quebrar algum lugar específico,
pode-se usar a tag <br>

Símbolos - Para colocar símbolos é usado & (o "E" ecomercial)
¥
¢

Emojis - Para emojis é usado = &#x (x minusculo). Obeservação: Após essa tag é colocado o codígo do emoji escolhido. Pode ser procurado acessando o site EMOJIPEDIA ✨
